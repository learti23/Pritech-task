# Pritech Task Manager

A clean and efficient task management application built with React Native and Expo. This app allows users to create, track, and manage their daily tasks with local persistence.

## Features
- **Task Persistence:** Tasks are saved locally using `AsyncStorage`.
- **CRUD Operations:** Create, Read, Update, and Delete tasks.
- **Clean UI:** User-friendly and intuitive interface.
- **TypeScript:** Developed with TypeScript for reliability.

## Installation
1. Clone the repository: `git clone <your-repository-url>`
2. Install dependencies: `npm install`
3. Run the app: `npx expo start`

## Technologies
React Native, Expo, AsyncStorage, TypeScript.
