# Pritech Task Manager

A clean and efficient task management application built with React Native and Expo. This app allows users to create, track, and manage their daily tasks with local persistence.

## Features
- **Task Persistence:** Tasks are saved locally using `AsyncStorage`, ensuring your data is safe even after closing the app.
- **CRUD Operations:** Easily Create, Read, Update, and Delete tasks.
- **Clean UI:** A user-friendly and intuitive interface designed for productivity.
- **TypeScript:** Developed with TypeScript to ensure code reliability and a better developer experience.

## Installation
If you were to run this locally:

1. Clone the repository: 
   `git clone <your-repository-url>`
2. Navigate to the folder:
   `cd pritech-task-manager`
3. Install dependencies: 
   `npm install`
4. Run the app: 
   `npx expo start`

## Technologies
- React Native
- Expo
- AsyncStorage
- TypeScript
